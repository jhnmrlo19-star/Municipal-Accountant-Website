    How to Edit This Website

    config.js — Edit site title, contact info, about text, services, Firebase credentials, and admin password.

    Admin Dashboard — Add, edit, or delete posts and view inquiries (no code needed).

    css/styles.css — Customize colors and visual styling.

    Firebase Setup

    1. Open Firebase Console
    2. Create or select your project
    3. Enable Realtime Database
    4. Copy your config values into config.js
    5. Set database rules to allow read/write (for development):
      { "rules": { ".read": true, ".write": true } }

    Important: Change the default admin password in config.js before going live!
